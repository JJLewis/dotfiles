## Quick Start

### Mac OS

- Open a terminal and run the following command:
  - `$ sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply jjlewis/dotfiles"`

- Launch 1Password
  - Go to Settings > Developer, and tick "Connect with 1Password CLI".
